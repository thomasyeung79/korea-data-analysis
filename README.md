# Korea Data Analysis Project

A data-driven exploration of South Korea’s global influence across:

- 📜 History
- 📊 Economy & System Efficiency
- 🎤 K-pop Industry
- ⚽ Sports Power

## Features

- Multi-page Streamlit app
- Interactive data visualizations
- Cultural + economic + sports analysis
- Optional embedded YouTube music (official sources)

## Run locally

pip install -r requirements.txt
streamlit run app.py

## Disclaimer
This project does not host or distribute copyrighted music. All audio content is embedded using official YouTube players.
